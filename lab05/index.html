Lab03 Reflection

September 20, 2025 • Topic: CSS Basics

During this Lab03 learning period, I have learned many new things about HTML and CSS. First and foremost, I learned how to use an external stylesheet to give structure and style to my HTML page in an organized way. To separate content from design and make the page easier to update, I learned how to link the style.css file in the <head> section. I used CSS to control spacing, heading styles, paragraphs, backgrounds, and text colors. I also created one ID selector for the header and footer and one class selector for the post card. After completing all of this, I validated both files using the W3C Validator to ensure there were no errors or warnings. Overall, through this module, I have learned more about semantic structure and making styles consistent, and I am confident that my HTML and CSS are accurate, readable, and meet professional standards.

Resources I used

  • MDN: Learn CSS
  • W3C CSS Validator
